# Política de Privacidade

Última atualização: 12 de junho de 2026

## Snake Game

O Snake Game foi criado como um jogo simples e direto. Esta política explica como o app lida com informações do usuário.

## Coleta de dados

O Snake Game não coleta, armazena, compartilha ou vende dados pessoais.

O app não solicita criação de conta, nome, e-mail, localização, contatos, fotos, identificadores de publicidade ou qualquer outra informação pessoal.

## Dados de jogo

O app pode salvar localmente no próprio dispositivo o recorde do jogador. Esse dado é usado apenas para exibir o melhor resultado dentro do jogo.

O recorde salvo localmente não é enviado para servidores, não é compartilhado com terceiros e pode ser removido ao apagar o app ou limpar os dados do dispositivo.

## Anúncios, analytics e rastreamento

O Snake Game não exibe anúncios.

O app não usa ferramentas de analytics.

O app não rastreia o usuário entre apps ou sites de terceiros.

## Compras no app

O Snake Game não possui compras dentro do app.

## Crianças

O app não coleta dados pessoais de crianças ou de qualquer outro usuário.

## Alterações nesta política

Esta política pode ser atualizada no futuro se o app passar a oferecer novos recursos. Quando isso acontecer, a data de atualização será alterada neste documento.
